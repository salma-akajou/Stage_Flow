# Charte Graphique : StageFlow

## 1. Identité Visuelle
L'identité de **StageFlow** repose sur le sérieux, la confiance et la modernité.

## 2. Palette de Couleurs
- **Primaire** : Indigo-600 (`#4f46e5`) - Utilisé pour les actions principales, les boutons et les états actifs.
- **Neutre** : Slate (900 pour le texte, 50 pour les fonds).
- **Statuts** :
    - Succès : Emerald-500
    - En attente : Amber-500
    - Refus : Rose-500

## 3. Typographie
- **Police Principale** : `Inter`, sans-serif.
- **Titres** : Bold, Inter.
- **Corps de texte** : Regular, Inter.

## 4. Composants Preline UI
Tous les composants doivent utiliser les classes Indigo-600 par défaut (ex: `bg-indigo-600`, `text-indigo-600`).
