# Stratégie de Contenu : StageFlow

## 1. Ton & Voix
- **Étudiant** : Clair, encourageant, simplifié.
- **Entreprise** : Professionnel, efficace, direct.

## 2. Structure par Page

### Landing Page
- **H1** : Trouvez votre prochain stage en quelques clics.
- **CTA** : Découvrir les offres / S'inscrire.

### Inscription
- **H1** : Rejoignez l'aventure StageFlow.
- **Option 1** : "Je suis un Étudiant" (Formulaire profil académique).
- **Option 2** : "Je suis une Entreprise" (Formulaire profil société).

### Dashboard Étudiant
- **H1** : Bonjour [Prénom], prêt pour de nouvelles opportunités ?
- **Stats** : Affichage clair du statut des candidatures (Accepté, Attente, Refusé).

### Gestion Entreprise
- **H1** : Vos offres de stage.
- **CTA** : Publier une nouvelle offre.
