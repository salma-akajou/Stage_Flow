# Wireframe : Espace Entreprise

## 1. Dashboard
- **Stats Cards** : Offres actives, Total Candidats, Candidats à traiter.
- **Graphique** : (Optionnel) Évolution des candidatures.

## 2. Gestion des Offres (CRUD)
- **Liste** : Tableau (Titre, Date, Vues, Candidats, Actions: Éditer/Supprimer).
- **Formulaire** : Champs pour créer/modifier une offre.

## 3. Gestion des Candidatures
- **Tableau** : Nom Candidat, Offre, Date, Statut (Dropdown pour changer : Accepté/Refusé/Attente).
- **Action** : Bouton [Voir Profil].

## 4. Profil Candidat
- **Vue Read-only** : Photo, Bio, CV, Liens Portfolio.
- **Actions** : [Contacter] | [Changer Statut].
