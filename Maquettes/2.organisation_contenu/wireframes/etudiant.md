# Wireframe : Espace Étudiant

## 1. Dashboard
- **Haut** : Stats Cards (Candidatures : Envoyées, En attente, Acceptées).
- **Milieu** : Liste "Activités récentes" (Derniers statuts mis à jour).

## 2. Liste des Stages
- **Haut** : Barre de recherche + Filtres (Ville, Durée, Domaine).
- **Grille** : Cards de stages (Logo, Titre, Entreprise, Ville, Durée, Badge Domaine).

## 3. Détails du Stage
- **Header** : Titre + Entreprise + Bouton [Postuler].
- **Contenu** : Description, Missions, Profil recherché, Localisation.

## 4. Mes Candidatures
- **Tableau** : Entreprise, Stage, Date, Statut (Badge couleur).
