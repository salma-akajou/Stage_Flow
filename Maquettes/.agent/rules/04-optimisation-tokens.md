# Règle 04 : Optimisation des Tokens

Les Design Tokens de **StageFlow** doivent être centralisés et cohérents.

- Ne pas utiliser de couleurs "adhoc" (ex: `#ff0000`). Utiliser les variables Tailwind (ex: `text-red-600`).
- Les espacements doivent suivre l'échelle Tailwind standard.
- Les bords arrondis doivent être `rounded-lg` par défaut pour un look premium.
