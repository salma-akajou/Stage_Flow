# Règle 03 : Qualité et Sécurité

Bien qu'il s'agisse de maquettes, les bonnes pratiques suivantes doivent être respectées :

- **Accessibilité** : Utiliser des labels ARIAs et des contrastes suffisants (standard Preline).
- **Responsive** : Approche Mobile-first systématique.
- **Validation HTML** : Structure sémantique correcte.
- **Performance** : Charger uniquement les scripts nécessaires.
