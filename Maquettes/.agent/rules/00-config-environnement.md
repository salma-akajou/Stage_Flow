# Règle 00 : Configuration de l'Environnement

Ce projet de maquettage pour **StageFlow** suit des standards stricts pour garantir la cohérence et la réutilisabilité.

## Environnement Technique
- **Framework CSS** : Preline UI (basé sur Tailwind CSS).
- **Icônes** : Lucide Icons.
- **Typographie** : Inter.
- **Couleurs** : Indigo-600 (Action), Slate (Texte/Fond).

## Standards de Fichiers
- Les atomes sont dans `3.maquettage/components/atoms/`.
- Les molécules sont dans `3.maquettage/components/molecules/`.
- Les mockups sont dans `3.maquettage/mockups/`.
- Toujours utiliser des chemins relatifs pour les ressources internes.
