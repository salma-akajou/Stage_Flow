# Protocoles de Workflow

Voici les étapes à suivre pour ajouter une nouvelle fonctionnalité ou page à la maquette.

## 1. Analyse
Vérifier l'impact dans `1.analyse_besoin/cahier_des_charges.md`.

## 2. Architecture
Mettre à jour le `2.organisation_contenu/sitemap.md`.

## 3. Composants
1. Créer les **Atomes** nécessaires s'ils n'existent pas.
2. Assembler les **Molécules**.
3. Déclarer la **Composition**.

## 4. Finalisation
Créer le **Mockup HTML** final en utilisant exclusivement les composants de la bibliothèque.
