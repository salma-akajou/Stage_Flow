# Stack Technique StageFlow

## Frontend
- **HTML5** : Structure sémantique.
- **Tailwind CSS** : Styles utilitaires (via CDN).
- **Preline UI** : Composants interactifs et complexes.
- **JavaScript Vanilla** : Logique de toggle et interactions simples.

## Assets
- **Lucide Icons** : Pack d'icônes cohérent.
- **Google Fonts (Inter)** : Typographie par défaut.

## Outils
- **UI Kit Gallery** : Dashboard interne pour explorer les composants.
