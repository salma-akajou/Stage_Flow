# Workflow : Création de Page

1. Choisir le nom du mockup (ex: `etudiant-dashboard.html`).
2. Créer le fichier HTML dans `3.maquettage/mockups/`.
3. Importer les styles standards.
4. Assembler les molécules existantes.
5. Ajouter au `mockups-manifest.md`.
