# Capacité : Analyse Métier

L'analyse métier consiste à comprendre les processus fondamentaux du projet StageFlow.

- **Étudiant** : Processus d'inscription -> Recherche -> Postulation -> Suivi.
- **Entreprise** : Processus de publication -> Tri des CV -> Entretien -> Acceptation.

Cette capacité assure que chaque composant UI répond à une étape réelle du métier.
