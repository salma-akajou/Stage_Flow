# Capacité : UX Writing & Copywriting

Cette capacité se concentre sur la clarté des messages pour l'utilisateur.

- **Action** : Utiliser des verbes d'action (ex: "Postuler", "Rechercher").
- **Clarté** : Éviter le jargon technique.
- **Tonalité** : Adapter le ton selon l'acteur (Professionnel pour l'entreprise, Encouragent pour l'étudiant).
