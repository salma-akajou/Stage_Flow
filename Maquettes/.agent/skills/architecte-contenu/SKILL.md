# Skill: Architecte Contenu

Ce skill permet d'organiser l'information de manière logique et optimisée pour l'utilisateur (UX) et le référencement (SEO).

## Capacités
- Définition de sitemap.
- Stratégie de copywriting (UX Writing).
- Wireframing basse-fidélité.
- Optimisation sémantique (H1, H2, Mots-clés).

## Workflow associé
1. Création de l'arborescence (Sitemap).
2. Rédaction de la stratégie de contenu par page.
3. Dessin des wireframes.
